## Nosso Objetivo

A proposta deste projeto foi criar uma solução direta e eficiente para o comércio online: **um catálogo digital de perfumes que elimina barreiras de compra.**

Diferente de e-commerces tradicionais complexos, nosso foco é a **agilidade**: permitir que o cliente escolha seu produto e finalize a negociação diretamente pelo **WhatsApp do vendedor**, sem a necessidade de criar contas, lembrar senhas ou preencher formulários extensos de pagamento no site.

---

## O Que Alcançamos (Entregas do Projeto)

Com a conclusão do desenvolvimento, o sistema hoje oferece uma infraestrutura completa para suportar esse modelo de negócios:

### Para o Vendedor (Lojista)
Entregamos um **ambiente administrativo seguro e exclusivo**. O vendedor tem total autonomia para:

- Acessar o sistema de forma protegida (login seguro).
- Gerenciar sua vitrine digital com facilidade.
- Cadastrar novos perfumes, controlar a quantidade em estoque, definir preços, descrições detalhadas, etc.
- Atualizar fotos e informações dos produtos em tempo real.

### Para o Cliente (Comprador)
Garantimos uma experiência de **vitrine aberta e transparente**:

- Acesso instantâneo ao catálogo de produtos (sem barreiras de login).
- Garantia de que as informações visualizadas (preço, disponibilidade) são confiáveis e validadas pelo sistema.
- Preparação do terreno para que a escolha dos produtos se transforme em uma mensagem pronta para o WhatsApp, facilitando o contato humano.

### Segurança e Confiabilidade
Embora invisível aos olhos do usuário comum, o sistema conta com diversas **"travas de segurança"** que impedem erros comuns, como cadastrar produtos com valores negativos, criar lojas duplicadas ou acessar dados de outros vendedores, garantindo a integridade do negócio.
